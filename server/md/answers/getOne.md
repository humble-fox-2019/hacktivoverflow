# Get All Answer

Used to get all Answer from server.

**URL** : `/answers/:id`

**Method** : `GET`



## Success Response

**Code** : `200 OK`

**Content example**

```json
{
    "_id": "",
    "answer": "Try to use console.log('Hello World!')",
    "questionId": "",
    "userId": "",
    "upvote": 0,
    "downvote": 0,
    "timestamps": {
      createdAt: "",
      updatedAt: ""
    }
}
```
